para instalar la libreria de Bluez (Bluetooth) se requiere lo siguiente:

1)  sudo apt-get install python3-dev
2)  sudo apt-get install blutooth
3)  sudo apt-get install libbluetooth-dev
Luego de ello probar instalando directamente el complemento

    sudo pip install pybluez
